# Capstone_3
South Region Sales Analysis with Power BI

## Capstone Description
This Power BI report analyzes sales performance for the EmporiUm 
South territory (Florida, South Carolina, and Texas) from 2022 to 
2025.

## Report Pages
- Page 1: Sales Trends Overview
- Page 2: State and Product Insights
- Page 3: Bonus Analysis (Store vs Online Sales)
